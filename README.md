## Library Management System 
In this project, a library is fully maintained with the help of python.
This library named **'PlayWithCode'** contained programming books.


## Objective of Library Management System
The main objective of Python Project on Library Management System is to manage the details of Books, User, Issuer. The purpose of the project is to build an application program to reduce the manual work for managing details online.


## Functionalities by python on Library Management System
* It tracks all the information of Books, User and Issues, etc.
* Manage the information of Books.
* It deals with the monitoring the information and transactions of issues/lend.
* This library also stored the data, i.e which book is lended by whom.
* User can lend the book, return the book, and can even add the book in library.
